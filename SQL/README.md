# SQL Scripts

## Overview

This folder contains all SQL scripts used in the E-Commerce Business Performance Analytics project. These queries were executed in PostgreSQL to perform business analysis, create reusable SQL views, generate KPIs and prepare data for Power BI dashboards.

---

## Database

- PostgreSQL

---

## SQL Scripts

| Script | Purpose |
|---------|---------|
| 01_average_delivery_time.sql | Calculates the average delivery time for customer orders. |
| 02_late_delivery_percentage.sql | Calculates the percentage of orders delivered late. |
| 03_monthly_order_trend.sql | Analyzes monthly order volume trends. |
| 04_monthly_revenue_trend.sql | Calculates monthly revenue generated over time. |
| 05_repeat_customer_analysis.sql | Identifies repeat customer purchasing behavior. |
| 06_revenue_by_payment_type.sql | Calculates revenue generated by each payment method. |
| 07_top_10_states_by_revenue.sql | Identifies the top-performing customer states by revenue. |
| 08_top_cities_by_revenue.sql | Identifies the highest revenue-generating cities. |
| 09_top_product_categories_by_revenue.sql | Calculates revenue contribution by product category. |
| 10_top_sellers_by_revenue.sql | Identifies the highest revenue-generating sellers. |
| 11_view_kpi_summary.sql | Creates a reusable KPI summary view. |
| 12_view_monthly_revenue.sql | Creates a reusable monthly revenue view. |
| 13_view_state_revenue.sql | Creates a reusable state revenue view. |
| 14_view_category_revenue.sql | Creates a reusable product category revenue view. |
