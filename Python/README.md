# Python Scripts

## Overview

This folder contains all Python scripts used in the E-Commerce Business Performance Analytics project. The scripts perform data cleaning, exploratory data analysis (EDA), dataset transformation, feature engineering, dataset merging, business analysis and visualization preparation before loading the data into PostgreSQL and Power BI.

The scripts are designed to be executed independently during different stages of the analytics workflow and use relative file paths for portability across systems.

---

## Libraries Used

- Pandas
- Matplotlib
- pathlib

---

## Script Execution Flow

| Script | Purpose |
|---------|---------|
| 01_data_cleaning.py | Cleans the customer dataset and performs initial data quality checks. |
| 02_orders_analysis.py | Processes order data and extracts monthly order trends. |
| 03_payments_analysis.py | Analyzes payment methods and payment values. |
| 04_product_analysis.py | Cleans product data and handles missing values. |
| 05_category_analysis.py | Analyzes product categories and order distribution. |
| 06_category_revenue_analysis.py | Calculates revenue generated by each product category. |
| 07_seller_analysis.py | Performs seller data quality analysis and duplicate checks. |
| 08_master_dataset.py | Merges all datasets into a single analytical dataset. |
| 09_revenue_trend_analysis.py | Calculates monthly revenue trends. |
| 10_revenue_trend_chart.py | Creates revenue trend visualizations using Matplotlib. |
| 11_state_revenue_analysis.py | Calculates revenue contribution by customer state. |
| 12_state_revenue_chart.py | Visualizes Top 10 states by revenue. |
| 13_seller_revenue_analysis.py | Identifies the highest revenue-generating sellers. |
| 14_seller_concentration_analysis.py | Measures revenue concentration among the Top 10 sellers. |
