# Dashboard Screenshots

## Overview

This folder contains screenshots of the Power BI dashboards developed for the **E-Commerce Business Performance Analytics** project.

These dashboards provide interactive visualizations for monitoring sales performance, customer behavior, product performance, seller analysis and strategic business insights.

---

# Dashboard Pages

| Dashboard | Description |
|-----------|-------------|
| Executive Dashboard | High-level KPIs including Revenue, Orders, Customers, Average Order Value and Delivery Performance. |
| Customer Analytics | Customer distribution, repeat customers, geographic analysis and purchasing behavior. |
| Product Analytics | Product category performance, seller contribution, Pareto analysis and revenue insights. |
| Sales & Order Intelligence | Monthly revenue, order trends, payment analysis and delivery performance. |
| Insights & Recommendations | Business observations, strategic recommendations and executive insights. |
