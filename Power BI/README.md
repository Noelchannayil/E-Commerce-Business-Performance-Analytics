# Power BI Dashboard

## Overview

This folder contains the Power BI dashboard developed for the E-Commerce Business Performance Analytics project. The dashboard transforms raw business data into interactive visualizations, enabling stakeholders to monitor business performance, customer behavior, product trends, sales performance and operational efficiency.

---

## Dashboard Pages

| Dashboard | Description |
|-----------|-------------|
| Executive Dashboard | High-level KPIs including Revenue, Orders, Customers, Average Order Value and Delivery Performance. |
| Customer Analytics | Customer distribution, repeat customers, geographic analysis and purchasing behavior. |
| Product Analytics | Product category performance, seller contribution, category revenue and Pareto analysis. |
| Sales & Order Intelligence | Revenue trends, monthly orders, payment methods, delivery performance and operational KPIs. |
| Insights & Recommendations | Business insights, strategic recommendations and key observations derived from the analysis. |
